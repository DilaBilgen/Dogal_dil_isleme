# Dogal_dil_isleme
# İnsan Kaynakları Projesi: CV-İlan Eşleşmesi'nde Metin Veri Ön İşleme ve Zipf Yasası Analizi
Proje Amacı
Bu projede, Cv-İlan metin verileri üzerinde Doğal Dil İşleme (NLP) teknikleri uygulanarak:
- Metinlerin temizlenmesi
- Dilsel dönüşümler yapılması
- Zipf Yasası analizi gerçekleştirilmesi
- Verinin modellemeye hazır hale getirilmesi
amaçlanmaktadır.
Ek olarak, metinler arası benzerlik ölçümü yapılarak basit bir eşleşme sistemi kurulmuştur.
